如何让你的hexo博客文章被浏览器收录

发表于 2025-12-26 分类于 教程
介绍如何让你的hexo博客文章被浏览器收录的方法。

在使用 hexo 搭建博客时，大家都希望自己的文章能够被浏览器收录，从而提高文章的曝光率和访问量。本文将介绍让你的文章被 bing 浏览器收录的方法。

1. 提交网站地图

网站地图(sitemap)是一个 XML 文件，列出了你网站上的所有重要页面，帮助搜索引擎更好地了解你的网站结构。你可以使用 hexo-generator-sitemap 与 hexo-generator-baidusitemap 插件来生成网站地图。

2. 提交给搜索引擎

将你的网站地图提交给搜索引擎，如 bing webmaster tools。这样可以加快搜索引擎对你网站的抓取和收录速度。

登陆 Bing Webmaster Tools ，添加你的网站并验证所有权。然后在“配置我的网站”中找到“提交站点地图”，输入你的网站地图 URL（例如：https://你的域名/sitemap.xml），点击提交。
为了防止是别人提交你的网站，提交之后你需要验证你的所有权。可以选择 html 文件验证或是添加元标签验证。

  1. HTML 文件验证：下载提供的 HTML 验证文件，将其放到你的 source 文件夹下，然后重新生成并部署博客，验证文件会被 hexo 自动放到根目录下。完成后返回 Bing Webmaster Tools 点击“验证”。

  2. 元标签验证：我所使用的 next 主题已经内置了元标签验证功能。在_config.next.yml 文件中找到 webmaster 部分，添加 Bing 提供的元标签内容。其他主题请自行查阅文档。

完成验证后，Bing 会开始抓取你的网站内容，并在搜索结果中显示你的文章。

3. 提高 SEO 的技巧

  1. 优化文章标题和描述：确保每篇文章都有独特且描述性的标题和元描述，有助于搜索引擎理解你的内容。
    在 scaffold 文件夹下的 post.md 文件中，你可以添加 description 字段。这等于是在模板中为每篇文章添加了元描述标签，此后每次新建文章时都会自动包含该字段。
