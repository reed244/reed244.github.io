astral，一个免费好用的游戏联机软件

发表于 2025-12-23 更新于 2025-12-26 分类于 教程
astral 是一个免费好用的游戏联机软件，支持 NAT 穿透和中继服务器，帮助玩家在广域网下实现局域网联机。

介绍

astral 是一个用于游戏联机的免费服务器软件。为了将广域网下的玩家连接到同一个局域网内进行游戏，astral 将先尝试通过 NAT 穿透技术直接连接玩家，如果无法直接连接，则通过 astral 的中继服务器进行转发。

在下载并安装好 astral 之后，打开软件。你将会看到四个选项：主页、房间、服务器和设置。

主页用于初始化玩家信息，如果你是使用 windows 版本移动要选择关闭防火墙，否则会导致无法联机。因为 astral 需要在后台运行一个服务器进程来处理联机请求。

房间用于创建或加入游戏房间。创建房间时，你可以设置房间名称、密码和最大玩家数量。加入房间时，你需要输入房间分享码或链接。

服务器用于选择连接的 astral 服务器节点。你可以手动选择节点以获得更好的连接质量(延迟越低越好)。

设置用于配置 astral 的各种选项，包括启动时自动运行、通知设置等。

使用方法

  1. 创建房间：点击“房间”选项卡，然后选择“创建房间”。填写房间名称、密码（可选）和最大玩家数量。点击“创建”按钮后，系统会生成一个房间分享码或链接，分享给其他玩家。

  2. 加入房间：点击“房间”选项卡，然后选择“加入房间”。输入房间分享码或链接，点击“加入”按钮即可进入房间。

  3. 选择服务器节点：点击“服务器”选项卡，浏览可用的 astral 服务器节点。选择一个延迟较低的节点以获得更好的连接质量。

  4. 连接: 在主页中点击“连接”按钮，astral 会尝试连接。成功后代表你们现在在同一个局域网内，可以开始游戏了。
