archlinux+niri 使用日记

发表于 2025-12-27 分类于 日记
记录我使用 archlinux 和 niri 的一些经历，防止遗忘。

出于对在 windows 中每日和爆红 C 盘斗争的厌倦，我决定尝试使用 linux 系统。而 archlinux + niri 正适合我。

QQ 无法视屏通话，无法使用中文输入法

我的 niri 是 wayland 环境，而 linuxqq 默认是 x11 环境。协议不同导致 qq 无法获得语音和输入法的权限，从而无法使用。修改.desktop 文件，将 qq 切换到 wayland 原生环境即可。
